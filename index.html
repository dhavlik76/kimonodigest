Instructions

  1. Install the Kimono Labs Chrome Extension or Bookmarklet.
  2. Visit any website to create a kimono API (something that changes often!)
  3. Make a selection for the article titles with property named title.
  4. Optionally make selections for the date submitted (named submitted), the author (author), the number of comments (comments), and finally the score or points (points) for each article.
  5. Do not change the collection name (i.e. keep collection1)
  6. Choose an update frequency of every 15 minutes.
  7. Create your API and copy the API ID (looks like a1b2c33c)
  8. Paste the API ID into to the form field above.
  9. Fill out the title of the source, the url of a square logo for it, and the hex color you'd like to use for the header, and you're done!
